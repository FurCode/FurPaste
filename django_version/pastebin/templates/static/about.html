About

Why the hell another pastebin?

Good question. Basically the world doesn't need another pastebin. There is pastie and paste.e-scribe.com which both use kick-ass highlighting libraries for highlighting the code and both have an initiutive user interface. Nevertheless I had too much free time and created this pastebin.

Feel free to not use it ;-)

Features

  • Tags
  • private pastes
  • web2.0 design ^^
  • different color schemes for the sourcecode
  • toggleable line numbers/syntax highlighting
  • reply to pastes with different highlighting
  • support for many python template languages
  • support for many scripting languages like python and ruby
  • XMLRPC support

Software Used

  • sqlite3 as database
  • django as framework
  • pygments for syntax highlighting
  • python as scripting language

Who?

mitsuhiko from the pocoo team is responsible for the pastebin. Pygments is a pocoo project led by Georg Brandl.
